cws
===

c websocket library

Testing
-------

In order to test, you'll have to install CUnit (http://cunit.sourceforge.net/).

`make test`

Coverage Report
---------------

In order to run coverage reports you'll have to install "lcov" package.

`make coverage`

Then, open `coverage/index.html`
